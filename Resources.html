Computer Hardware

Hardware resources are the physical components of a computer system. These include devices
that you can see and touch, such as the CPU (Central Processing Unit), monitor,
keyboard, mouse, hard disk, and printer. These components work together to perform
computing tasks efficiently.

Computer Software

Software resources are programs and applications that run on a computer.
They include operating systems like Windows and Linux, as well as
application software such as MS Word, Excel, web browsers, and
antivirus programs. Software helps users perform specific tasks easily.
